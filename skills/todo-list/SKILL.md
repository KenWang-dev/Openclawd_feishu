# 待办/购物清单

将原始文本转换成结构化的待办清单或购物清单。

## 输入

用户发送的原始内容（可以是购物物品、任务事项等）

## 处理

1. 解析每一条内容
2. 识别分类（可选）
3. 格式化为清单

## 输出格式

📋 **待办清单**：

- [ ] 项目1
- [ ] 项目2
- [ ] 项目3

或

🛒 **购物清单**：

- [ ] 项目1
- [ ] 项目2
- [ ] 项目3

## 示例

用户输入：
```
苹果 5个
牛奶 2盒
鸡蛋 1盒
面包
```

输出：
🛒 **购物清单**：

- [ ] 苹果 5个
- [ ] 牛奶 2盒
- [ ] 鸡蛋 1盒
- [ ] 面包
