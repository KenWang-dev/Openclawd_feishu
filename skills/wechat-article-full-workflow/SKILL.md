# 微信公众号文章全流程工作流（已固化）

## 概述

整合「文章初稿生成」+「AI配图」+「公众号发布」的完整工作流。

**核心理念**：每个阶段必须用户确认后才能进入下一步（断点机制）

---

## 三阶段 + 断点流程

```
用户发送素材/文档
        ↓
    【断点1】询问意图 ← 必须先问用户要做什么
        ↓
第一阶段：输入 + 研究
        ↓
第二阶段：生成初稿（纯文本）
    Alpha → Beta → Omega → Gamma
        ↓
    【断点2】用户确认初稿 ← 必须等用户OK
        ↓
第三阶段：配图 + 发布
    封面配图 → 文章配图(不同prompt) → 发草稿箱
        ↓
    【断点3】用户确认发布 ← 必须等用户OK
```

## 完整工作流详解

### 第一阶段：输入 + 研究

1. 用户提供素材/想法/文档
2. 保存到 `wechat-article-workflow/input/YYYY-MM-DD-主题.md`
3. 可选：调用 Tavily 深度搜索补充背景信息

### 第二阶段：生成初稿（4步）

使用 `wechat-article-workflow` 的 4 个 prompt：

| 阶段 | 文件 | 作用 |
|------|------|------|
| Alpha | Alpha-初稿改写.md | 基于素材生成初稿 |
| Beta | Beta-评审.md | 评审初稿问题 |
| Omega | Omega-修改.md | 根据评审修改 |
| Gamma | Gamma-验收.md | 最终验收 |

**输出**：`wechat-article-workflow/output/终稿-YYYY-MM-DD.md`（纯文本）

### 第三阶段：配图 + 发布

#### 配图配置

**Aiberm API 配置**（已验证可用）：
```
AIBERM_API_KEY=sk-MaotlYGe8iI54E3JJpECfK0aTk6gTQvt0NQ9aalnZ1GMvglp
AIBERM_BASE_URL=https://aiberm.com/v1
Model: gemini-3-pro-image-preview
```

#### 配图流程（关键！）

1. **封面图**：1张，16:9 或 2.35:1
2. **文章插图**：2张，**必须用不同prompt生成！**
   - 插图1：文章第一部分相关内容
   - 插图2：文章第二部分相关内容
3. **上传到微信公众号素材库**：用 `material/add_material` 接口
4. **替换HTML中的图片URL**：用微信返回的永久URL

#### 关键代码

```bash
# 生成图片（必须用不同prompt！）
curl -s "https://aiberm.com/v1beta/models/gemini-3-pro-image-preview:generateContent" \
  -H "Authorization: Bearer $AIBERM_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [{"parts": [{"text": "你的prompt1"}]}],
    "generationConfig": {"responseModalities": ["IMAGE"], "aspectRatio": "16:9", "imageSize": "1K"}
  }'

# 上传到微信（用永久素材接口！）
curl -F "media=@image.png" "https://api.weixin.qq.com/cgi-bin/material/add_material?access_token=$TOKEN&type=image"
```

## 常见问题与解决

| 问题 | 原因 | 解决 |
|------|------|------|
| 插图重复 | 两次生成用了相同或相似prompt | 每次生成确保prompt明确区分内容 |
| 图片不显示 | 用的是临时素材URL | 必须用 `material/add_material` 获取永久URL |
| 图片变成缩略图 | URL中包含特殊字符未转义 | 确保URL正确，用https |

## 强制规则

1. **永远不断点**：每个阶段完成后必须等用户确认
2. **先问再做**：收到任何文档，第一时间问用户意图
3. **插图不同**：每张插图必须用不同prompt生成
4. **永久URL**：图片必须上传到微信永久素材库
5. **用户优先**：用户说"先问问我的想法" → 立即停止并询问
