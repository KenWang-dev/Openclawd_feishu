# 飞书消息排版指南

## 加粗语法

✅ **支持**：`**文字**`（Markdown 标准语法）

❌ **不支持**：无

**注意**：必须测试后再大规模发送，曾有 target ID 错误导致 400 报错。

## 空行/换行

✅ **支持**：全角空格 `　`（U+3000）

❌ **不支持**：连续 `\n\n` 会被飞书压缩

**用法**：在需要空行的地方输入一个全角空格

## 列表格式

```
1. 项目1 | 分类 | ⭐分数
描述内容

　（空行）

2. 项目2 | 分类 | ⭐分数
描述内容
```

## 分割线

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| 400 Bad Request | target ID 错误或语法不支持 | 测试单条消息排查 |
| 空行被压缩 | 使用普通空格或连续换行 | 用全角空格 `　` |
| 加粗不生效 | 飞书企业版可能有差异 | 先测试 `**test**` |
