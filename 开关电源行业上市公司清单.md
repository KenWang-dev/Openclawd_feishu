# 开关电源行业上市公司清单

## 一、LED驱动电源细分领域

### 1. 英飞特（300582.SZ）
- **全称**：英飞特电子（杭州）股份有限公司
- **主营业务**：LED驱动电源、LED照明配件
- **市场地位**：全球LED驱动电源领先企业
- **特点**：技术领先，海外市场占比高
- **对标价值**：★★★★★（与普德新星最相似）

### 2. 茂硕电源（002660.SZ）
- **全称**：茂硕电源科技股份有限公司
- **主营业务**：开关电源、LED驱动电源、光伏逆变器
- **市场地位**：深圳本土老牌电源企业
- **特点**：产品多元化，消费电子+工业+光伏
- **对标价值**：★★★★☆（深圳本土同行）

### 3. 伊戈尔（002922.SZ）
- **全称**：伊戈尔电气股份有限公司
- **主营业务**：LED驱动电源、电源变压器
- **市场地位**：LED照明驱动头部企业
- **特点**：专注照明领域

### 4. 光莆股份（300632.SZ）
- **全称**：厦门光莆电子股份有限公司
- **主营业务**：LED照明、LED驱动电源、FPC
- **特点**：LED产业链一体化

## 分析维度建议

对比分析时，重点关注以下指标：

| 维度 | 关键指标 | 数据来源 |
|------|----------|----------|
| **财务健康** | 营收、利润、现金流、毛利率 | 财报 |
| **成长性** | 营收增速、净利润增速 | 财报 |
| **市场地位** | 市场份额、行业排名 | 研报、行业协会 |
| **技术能力** | 研发投入、专利数量 | 财报、专利局 |
| **客户结构** | 前五大客户占比 | 年报、招股说明书 |
| **盈利模式** | OEM/ODM/自有品牌占比 | 年报、研报 |
| **产品结构** | 各产品线营收占比 | 年报 |
