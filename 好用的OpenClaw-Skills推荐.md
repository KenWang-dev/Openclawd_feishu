# 好用的 OpenClaw Skills 推荐列表

## ✅ 已安装技能

### 1. Self-Improving Agent (v1.1.0) ⭐⭐⭐⭐⭐
- **功能**：从错误中学习，持续优化记忆
- **用途**：
  - 自动记住你的偏好和纠正
  - 跨会话累积知识
  - 三层记忆架构（热/温/冷存储）
- **推荐指数**：⭐⭐⭐⭐⭐
- **适合**：所有用户，特别是长期使用者

### 2. Agent Browser (v0.2.0) ⭐⭐⭐⭐
- **功能**：浏览器自动化，网页交互
- **用途**：
  - 自动化网页操作
  - 快照和分析页面
  - 点击、输入、导航
- **推荐指数**：⭐⭐⭐⭐
- **适合**：需要自动化网页任务的用户

### 3. Tavily Search (手动安装) ⭐⭐⭐⭐⭐
- **功能**：AI 优化的网页搜索
- **用途**：
  - 实时搜索最新信息
  - 提取网页内容
  - 深度研究话题
- **推荐指数**：⭐⭐⭐⭐⭐
- **适合**：所有用户（需要 API Key）
